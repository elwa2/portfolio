# منصة نُور التعليمية — وصف المشروع

## نظرة عامة

منصة تعليمية عربية كاملة لعرض وإدارة الكورسات، مبنية بـ HTML/CSS/JS خالص في ملف واحد، مع بنية بيانات جاهزة للربط بقاعدة بيانات SQLite محلية.

---

## التصميم البصري

| العنصر | التفصيل |
|---|---|
| الاتجاه | Nocturnal Glass — خلفية داكنة عميقة + لوحات زجاجية شفافة |
| الألوان الرئيسية | أخضر فيروزي `#2DD4BF` وأزرق سماوي `#38BDF8` |
| الخط | Cairo (Google Fonts) — عربي احترافي |
| الاتجاه النصي | RTL كامل |
| الأيقونات | إيموجي دائرية كبيرة وواضحة |

---

## صفحات المنصة

### ١. الرئيسية `home`
- **Hero Strip** ترحيبي مع تأثير توهج متحرك
- **إحصائيات المنصة**: عدد الكورسات، المدرّسين، الشهادات، التقييم
- **متابعة التعلم**: الكورسات التي بدأها المستخدم
- **الأكثر شعبية**: كورسات مقترحة

### ٢. الاستكشاف `explore`
- شرائح فئات قابلة للتصفية (برمجة، تصميم، ذكاء اصطناعي…)
- شبكة كاملة لجميع الكورسات
- تصفية ديناميكية فورية

### ٣. كورساتي `my-courses`
- عرض الكورسات المسجّل فيها فقط
- شريط تقدم مرئي لكل كورس

### ٤. المفضلة `favorites`
- قائمة الكورسات المحفوظة
- إزالة من المفضلة مباشرة

### ٥. الملف الشخصي `profile`
- إحصائيات الطالب الشخصية
- سجل آخر النشاطات

## المكوّنات الرئيسية

```
TopBar       — شريط علوي ثابت: لوغو + بحث + إجراءات
Sidebar      — شريط جانبي: قائمة + تقدم أسبوعي
CourseCard   — بطاقة كورس: صورة + تقدم + معلومات
Modal        — نافذة تفاصيل الكورس + التسجيل
Toast        — إشعارات خفيفة (إضافة مفضلة / تسجيل)
```

---

## بنية قاعدة البيانات (SQLite — جاهزة للربط)

```sql
CREATE TABLE courses (
  id          INTEGER PRIMARY KEY,
  title       TEXT NOT NULL,
  category    TEXT,
  emoji       TEXT,
  description TEXT,
  instructor  TEXT,
  duration    TEXT,
  lessons     INTEGER,
  level       TEXT CHECK(level IN ('beginner','inter','adv')),
  rating      REAL,
  progress    INTEGER DEFAULT 0,
  enrolled    INTEGER DEFAULT 0,
  featured    INTEGER DEFAULT 0
);

CREATE TABLE favorites (
  user_id   INTEGER,
  course_id INTEGER REFERENCES courses(id),
  saved_at  DATETIME DEFAULT CURRENT_TIMESTAMP,
  PRIMARY KEY (user_id, course_id)
);

CREATE TABLE users (
  id       INTEGER PRIMARY KEY,
  name     TEXT,
  initials TEXT,
  email    TEXT UNIQUE
);
```

نقاط الربط في الكود محدّدة بتعليق `/* DB-READY */`.

## الانتقالات والحركات

| الحركة | الاستخدام |
|---|---|
| `fadeSlideUp` | دخول البطاقات والأقسام |
| `scaleIn` | ظهور بطاقات الكورسات |
| `heartBeat` | زر المفضلة عند الضغط |
| `orbitFloat` | الـ stat pills في الـ Hero |
| `pulseGlow` | توهج أيقونات الإحصائيات |
| `bgPulse` | الخلفية المتحركة في الـ Hero |

---

## الميزات التفاعلية

- **المفضلة**: إضافة/إزالة مع حفظ في الـ state
- **البحث**: تصفية ديناميكية فورية
- **التنقل**: نظام صفحات بدون إعادة تحميل
- **الـ Modal**: تفاصيل كاملة + تسجيل
- **الـ Toast**: إشعارات خفيفة للعمليات
- **الـ Chip Filter**: تصفية بالفئة في صفحة الاستكشاف
