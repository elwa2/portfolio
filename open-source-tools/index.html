الأدوات مفتوحة المصدر

استكشف مجموعتنا المتنوعة من الأدوات التي نطورها لمساعدتك في مشاريعك البرمجية والتصميمية.

مجموعة أدوات المطورين

أداة شاملة لضغط أكواد CSS، تعمية ملفات JavaScript، وتشفير ملفات SVG للاستخدام في الويب.

أداة تهيئة صور المقالات

أداة متقدمة لتهيئة صور المقالات بمقاس 1200×628، معالجة الصورة، إضافة بادج الموقع، تصدير WebP، توليد اسم ملف سيو، معاينة الحجم والجودة، مع حفظ آخر صورة تلقائياً ودعم لصق الصور (Ctrl+V).

JavaScript Comment Cleaner

أزل تعليقات JavaScript فقط بشكل آمن واحترافي، مع الحفاظ على النصوص وTemplate Literals وRegex، ودعم نسخ/تنزيل النتيجة ورفع ملفات JS.

محول MP4 إلى JPG

حوّل فيديو MP4 إلى صور JPG عالية الجودة بسهولة وسرعة. يدعم استخراج جميع الإطارات أو إطار واحد فقط مع إمكانية تحميل جميع الصور دفعة واحدة.

تغيير حجم الصور

قم بتعديل أبعاد صورك بدقة سواء بالبكسل أو بالنسبة المئوية مع الحفاظ على الجودة.

A4 Photo Grid Studio

ارفع فولدر صور وحوّله تلقائياً إلى شبكة A4 بأحجام وأشكال مختلفة (كبير/صغير/دائري)، ثم نزّل التصميم PNG أو انسخه مباشرة إلى Illustrator.

منشئ التدرجات اللونية

صمم تدرجات عصرية (Linear Gradients) لمشاريعك مع دعم التوليد العشوائي والنسخ السريع.

أدوات الألوان الذكية

منتقي ألوان متطور يعرض قيم HEX, RGB، التشبع والسطوع، مع اقتراح لوحات لونية متناسقة.

مولد متغيرات اللون الرئيسي

أدخل اللون الأساسي لتوليد متغيرات CSS جاهزة مثل --color-primary ونسخ درجات فاتحة/غامقة/عكسية مع كود :root كامل.

مختبر نظام الألوان الرئيسي

أدخل اللون الرئيسي لتحصل فوراً على درجات فاتحة/غامقة، ألوان متناغمة، ومعاينة مباشرة للواجهة مع كود :root + JSON جاهز للنسخ والتنزيل.

Mistral OCR Studio

ارفع صورة أو PDF واستخرج النص عبر Mistral OCR API مع معاينة الصفحات، محرر نص مباشر، نسخ، تنزيل TXT، وطباعة داخل نفس الأداة.

مولد رموز QR

حول روابطك أو نصوصك إلى رموز QR فورية عالية الجودة قابلة للتحميل والاستخدام.

مولد كلمات المرور

أنشئ كلمات مرور معقدة وآمنة تماماً مع خيارات تخصيص للطول ونوع الأحرف.

محول WebP وضاغط الصور

حول صورك إلى تنسيق WebP الخفيف والمثالي للويب لزيادة سرعة تحميل موقعك.

محول روابط GitHub للـ CDN

حول روابط ملفاتك في GitHub إلى روابط CDN سريعة عبر خدمة jsDelivr بضغطة واحدة.

روابط مفيدة للمصممين

مجموعة مختارة من أهم المواقع والروابط التي تهم المصممين والمطورين (أدوات AI، أكواد CSS، وغيرها).

مولد تأثير الزجاج

صمم واجهات عصرية بأسلوب Glassmorphism التفاعلي واحصد كود CSS جاهز لمشاريعك مع معاينة مباشرة.

أدوات KDP للناشرين

مجموعة متكاملة لإنشاء كتب التلوين، ألغاز السودوكو، والتصاميم الداخلية الجاهزة للنشر على أمازون.

Base64 Encoder & Decoder

أداة سهلة لتحويل النصوص والملفات من وإلى Base64 مع دعم عمليات التشفير والفك بسرعة عالية.

MegaSplit - تقسيم الصور

قم بتقسيم صورك الكبيرة والطويلة إلى عدة أجزاء بدقة عالية وبدون فقدان الجودة. مثالية لصور المنتجات والبانرات!

File Toolkit - مجموعة أدوات الملفات

مجموعة شاملة ومتكاملة من الأدوات لمعالجة وتحرير الصور بالفلاتر، تحويل الصيغ، الضغط، وأدوات PDF الأساسية.

Image Optimizer - محسّن الصور المتقدم

أداة احترافية لضغط وتحسين الصور بتقنية محلية 100%. ضغط ذكي، تحويل الصيغ، تغيير الأبعاد، ومقارنة Before/After التفاعلية.

Smart Image Compressor - ضغط ذكي للصور

اضغط ملفات كبيرة (مثل 7MB) للوصول إلى أقل من 1MB مع خيار نفس الصيغة PNG أو التحويل بين الصيغ، وتقليل الدقة من 300 إلى 150 أو 70 بسهولة.

Pro Image Editor - محرر الصور الاحترافي

محرر صور متقدم داخل المتصفح: تحكم RGB، Vignette، ضوضاء، Blur/Sharpen، تدوير وقلب، أداة قص بالسحب، مقارنة Before/After، وتنزيل PNG بجودة عالية.

مصمم متخصص في تطوير وتصميم متاجر منصة زد وسلة، أقدم خدمات تخصيص CSS لجعل متجرك فريداً ومميز. خبرة واسعة في تحسين واجهة المستخدم وتجربة التسوق لزيادة المبيعات.
